# Проект: Прохождение частиц через изогнутую трубу

Цель проекта: разработать модель движения частиц через S-образный коридор и подобрать параметры внешнего поля так, чтобы все частицы переходили из одного конца в другой за минимальное время.

## Текущее состояние проекта
Сейчас реализована первая итерация: модель идеального 2D газа в прямоугольной области. Частицы движутся с начальными случайными скоростями, отражаются от стенок и отображаются в окне SFML.

## Глобальная цель
Дальнейшая цель проекта: развить текущую модель в кроссплатформенное приложение на C++ с SFML, добавив межмолекулярное взаимодействие, внешнее потенциальное поле и S-образный коридор.

## Дорожная карта

Проект развивается по итерациям. На данный момент существуют итерации:

1. `iter-1`: идеальный 2D газ в прямоугольной плоскости.
2. `iter-2`: добавляются межмолекулярное взаимодействие, внешнее поле и S-образный коридор.

>  **Как переключиться на версию конкретной итерации:**
> `git switch --detach iter-1` (для первой версии) или `git switch main` (для последней стабильной).

## Инструкция по запуску

### Предварительные требования
Для компиляции необходима библиотека **SFML**.
- **Linux:** `sudo apt install libsfml-dev`
- **Windows:** Скачать архив с [sfml-dev.org](https://www.sfml-dev.org/download/sfml/) (версия под ваш MSVC) и распаковать (например, в `C:\Libs\SFML`).

---

### 🔹 Вариант А: Linux (Ubuntu/Debian)

```bash
# 1. Клонирование (если еще не сделано)
git clone <URL_РЕПОЗИТОРИЯ>
cd <ПАПКА_ПРОЕКТА>

# 2. Установка зависимостей
sudo apt update && sudo apt install g++ cmake make libsfml-dev

# 3. Выбор итерации (опционально)
# git switch --detach iter-1  # Для запуска 1-й итерации

# 4. Сборка
mkdir build && cd build
cmake ..
make

# 5. Запуск
./simulation
```

### 🔹 Вариант Б: Windows (Visual Studio 2019/2022)

#### Шаг 1: Подготовка SFML
1. Скачайте версию SFML, соответствующую вашей версии Visual Studio (MSVC 16.0 для VS2019, 17.0 для VS2022) и разрядности (x64).
2. Распакуйте в путь без пробелов, например: `C:\Libs\SFML`.

#### Шаг 2: Открытие в Visual Studio
1. Запустите VS -> **"Открыть локальную папку"** (Open a local folder).
2. Выберите корень проекта (где лежит `CMakeLists.txt`).
3. VS автоматически настроит CMake.

#### Шаг 3: Настройка путей (если CMake не нашел SFML)
1. Меню **Проект** -> **Настроить параметры CMake** (Configure CMake Settings).
2. В открывшемся `CMakeSettings.json` найдите секцию `"configurations"` и добавьте строку:
   ```json
   "cmakeCommandArgs": "-DSFML_DIR=C:/Libs/SFML/lib/cmake/SFML"
   ```
   *(Замените путь на ваш реальный путь к папке SFML)*.
3. Сохраните файл.

#### Шаг 4: DLL и запуск
1. После сборки найдите `.exe` в папке `out/build/x64-Debug/`.
2. Скопируйте все `.dll` файлы из `C:\Libs\SFML\bin` в эту же папку (`out/build/x64-Debug/`).
   - Нужны: `sfml-graphics-2.dll`, `sfml-window-2.dll`, `sfml-system-2.dll`.
3. Запустите `simulation.exe` через VS (правой кнопкой -> Запуск) или напрямую.

**Управление в программе:**
- Изменение размера окна: Пропорции сохраняются автоматически (черные поля).
- Перезапуск частиц: Клавиша `SPACE`.
- Выход: Закрытие окна.

## Что уже реализовано в `iter-1`
- Генерация частиц в прямоугольной области.
- Случайная начальная скорость для каждой частицы.
- Отражение от границ с сохранением модуля скорости.
- Масштабирование окна без искажения сцены.

## Что пока не реализовано
- Столкновения частиц друг с другом.
- Межмолекулярное взаимодействие.
- Внешнее поле.
- S-образный коридор.
